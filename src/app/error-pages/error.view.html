There was an unexpected error, please check the console for more information.

While retrieving the application context angular2-carbonldp returned an HTTP.Errors.UnknownError.

This usually means that the server response didn't include the necessary CORS headers. 90% of the time, it's because the application you are trying to retrieve doesn't exist.

Double check that your application exists

Other common reasons are:

  • The configured Carbon platform host is wrong (right now it's set to {{ carbon.getSetting( "domain" ) }})
  • The platform server doesn't support https and you didn't turn it off (use carbon.setSetting( 'http.ssl', false );)
  • The platform server is down or not accessible (open {{carbon.resolve( "api/" )}} in the browser to test it)
